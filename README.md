## 🚀 Deploy to GitHub Pages

1. Create repo named **`dawaikart247.com`**
2. Upload `index.html`, `README.md`, and the entire `frames/` folder
3. Go to **Settings → Pages → Source: main / root**
4. Add custom domain: `dawaikart247.com`
5. Configure Cloudflare DNS (see below)
